## 🧠 About

**AcadeVault** is a secure academic document vault built for students. Store and organize your 10th marksheet, Intermediate marksheet, all 8 B.Tech semester results, and certifications — all in one private, password-protected dashboard. No app to install, no backend required — just open the link and get started.

> Built by a student, for students. Never lose an important marksheet again.

## ✨ Features

### 🔐 Secure Login
- Create your own account with email & password
- Each user gets a completely private vault
- Your data is encrypted & protected — only you can access it

### 📊 Dashboard
- Total documents stored counter
- Vault completion percentage
- Semesters uploaded tracker
- Certifications count
- Quick-access cards for all sections

### 📁 Document Sections

| Section | What to Upload |
|---|---|
| 📘 10th Marksheet | SSC / CBSE / State Board result |
| 📗 Intermediate | HSC / +2 marksheet |
| 🎓 B.Tech Sem 1–8 | Individual result for each semester |
| 🏅 Certifications | NPTEL, Coursera, internship, any certificate |

### 👤 My Profile
- Upload your profile photo
- Personal details — name, DOB, gender, blood group, bio
- Contact info — phone, WhatsApp, address
- Academic details — college, department, roll number, admission number, CGPA
- Social links — LinkedIn, GitHub, portfolio website
- Resume upload with download button
- Skills & technologies field
- All details update live in the profile hero card

### 🔗 Share Access
- Generate a shareable link for your documents
- Choose which sections to share
- Invite friends to create their own AcadeVault

### 🌌 UI Design
- Deep space dark background with aurora glow animations
- 120-star twinkling star field
- Glassmorphism cards with backdrop blur
- Fully mobile responsive
- Custom AcadeVault logo by Vignesh Yadala
- Your friends can visit the same link, create their own account, and store their documents privately — completely separate from yours.
